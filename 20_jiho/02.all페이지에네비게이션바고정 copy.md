네비게이션 바 z-
index 수정 (수정된 계획)

문제 분석
style.css 파일의 .perspective-wrap:hover { z-index: 9999 !important; } 규칙 때문에 카드에 마우스를 올리면 z-index: 9999가 설정됩니다. 네비게이션 바가 z-50 또는 z-60이면 카드가 네비게이션 바를 가리게 됩니다.

해결 방법
네비게이션 바의 z-index를 9999보다 높게 설정하여 항상 최상단에 위치하도록 합니다. 네비게이션 바를 `z-[10000]` 또는 `z-[100]`으로 설정합니다.

변경 사항
templates/nav.html: 네비게이션 바의 z-index 클래스를 z-50에서 `z-[10000]`으로 변경 (1줄)
